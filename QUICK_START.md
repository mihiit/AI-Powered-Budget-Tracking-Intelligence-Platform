## Demo Features Tour

### 1️⃣ **Dashboard** (Main Overview)

**What you'll see:**
- 💰 **Net Worth Card** - Current balance with income/expense breakdown
- 📊 **Income vs Expense Chart** - 6-month trend visualization
- 🍕 **Category Breakdown** - Donut chart showing spending by category

- ⚠️ **Cash Flow Risk Badge** - Real-time risk assessment

**Try this:**
- Check your Financial Health Score breakdown
- See which categories you spend most on
- Review the risk assessment

### 2️⃣ **Transactions** (Smart Categorization)

**What you'll see:**
- 📝 **90 days** of realistic mock transactions
- 🤖 **ML confidence scores** (e.g., "92% confident this is Food & Dining")
- ⚠️ **Anomaly alerts** for unusual transactions
- 🏷️ **Category tags** with color coding

**Try this:**
1. Click **"Add Transaction"**
2. Fill in details (e.g., "Starbucks Coffee", $5.50, Food & Dining)
3. Click **"Add Transaction"** button
4. See your new transaction appear at the top with ML metadata

**Pro Tip:** The system auto-categorizes based on description keywords!

### 4️⃣ **Goals** (Success Probability Calculator)

**What you'll see:**
- 🎯 **2 active goals** (Emergency Fund, MacBook Pro)
- 📈 **Progress tracking** with milestones
- 🧮 **AI projections:**
  - Required monthly savings
  - Current savings pace
  - **Success probability** (e.g., "62% chance of reaching goal")
- 📊 **Confidence intervals** (optimistic/pessimistic scenarios)

**Try this:**
- Check the **Emergency Fund** goal
- See success probability (should be 62%)
- Read the AI recommendation below

**Understanding the AI:**
- If success probability < 70%, you'll see a recommendation
- Example: "Increase monthly savings by $130 to improve success rate"

### 5️⃣ **Insights** (Behavioral AI)

**What you'll see:**
- 📈 **6-month balance forecast** (line chart)
- 💡 **3 personalized insights:**
  1. **Weekend Overspending** - "You spend 2x more on weekends"
  2. **Unusual Transaction Alert** - "$450 shopping transaction detected"
  3. **Budget Risk** - "Transportation budget at risk"
- 💰 **Estimated savings** per recommendation

**Try this:**
1. Read the "Weekend Overspending" insight
2. Check the data (weekend avg vs. weekday avg)
3. See the estimated savings ($692/month)
4. Click the ❌ button to dismiss an insight

**AI Explanation:**
Each insight includes:
- Pattern detected (what the AI found)
- Supporting data (numbers to back it up)
- Recommendation (how to fix it)
- Estimated impact (savings in dollars)

## FAQ

**Q: Is this real data?**  
A: No, all data is mock/generated for demonstration purposes.

**Q: Does it connect to my bank?**  
A: No, this is a frontend demo. Bank integration requires backend API (see ARCHITECTURE.md).

**Q: Are the ML predictions real?**  
A: The ML algorithms are real (documented in ARCHITECTURE.md), but running on mock data.

**Q: Can I save my data?**  
A: Data is stored in browser localStorage. Clearing browser data will reset to demo state.

**Q: What languages are supported?**  
A: English and Hindi (partial translation for demonstration).

**Q: Is dark mode saved?**  
A: Yes, theme preference is saved to localStorage.
